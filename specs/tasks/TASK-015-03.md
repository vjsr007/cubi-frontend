## Description
Complete the IGDB stub in `igdb_service.rs`. Implement Twitch OAuth2 client credentials flow, token caching, and game search with metadata + media extraction. Covers all PC games including Epic, GOG, EA.

## Acceptance Criteria
- [ ] `IgdbService` struct with `client_id`, `client_secret`, in-memory token cache (`Arc<Mutex<Option<IgdbToken>>>`)
- [ ] `authenticate() -> Result<(), String>` fetches token from `https://id.twitch.tv/oauth2/token`
- [ ] Token auto-refreshed on 401 response
- [ ] `search_game(title: &str) -> Result<Option<IgdbGameData>, String>` queries `https://api.igdb.com/v4/games` with APIcalypse body
- [ ] Query fields: name, summary, genres.name, first_release_date, involved_companies.company.name, involved_companies.developer, involved_companies.publisher, rating, screenshots.url, artworks.url, cover.url, websites.url, websites.category, videos.video_id
- [ ] Platform filter: `platforms = (6)` (PC Windows)
- [ ] Title normalization for matching: lowercase, remove articles (the/a/an), strip punctuation
- [ ] Similarity scoring: exact match preferred, then contains match
- [ ] Screenshot URLs: replace `t_thumb` with `t_1080p`, prepend `https:`
- [ ] Cover URL: replace `t_thumb` with `t_cover_big`, prepend `https:`
- [ ] `download_igdb_media(data, dest_dir, game_name)` downloads cover + up to 3 screenshots
- [ ] `IgdbGameData` struct with all extracted fields
- [ ] Unit test: mock auth + game search response
