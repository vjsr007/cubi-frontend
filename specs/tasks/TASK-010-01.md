# TASK-010-01: Create GameBoxCase 3D Component

## Context
Build a reusable 3D CSS game box case component that displays front and back faces with a flip-on-click animation. Uses CSS `perspective`, `rotateY`, `backface-visibility: hidden`, and `transform-style: preserve-3d`.

## Parent Design
- **DES**: [DES-010 — Game Box 3D Detail View](../design/DES-010-game-box-detail-view.md)

## Acceptance Criteria
- [ ] Component renders a 3D box with front face, back face, left spine, right spine
- [ ] Clicking the box toggles flip state (rotateY 0 ↔ 180deg)
- [ ] Front face shows box_art image or styled placeholder with game title
- [ ] Back face shows back_cover image or metadata panel
- [ ] Spine sides show system-colored strips
- [ ] Flip animation is 600ms ease-in-out, GPU-accelerated
- [ ] Component accepts `flipped` and `onFlip` props for controlled mode, or manages its own state

## Implementation Notes

### Files to Create/Modify
| File | Action | Description |
|------|--------|-------------|
| `src/components/library/GameBoxCase.tsx` | CREATE | New 3D box case component |

### Key Code Patterns
```tsx
// CSS 3D flip pattern
<div style={{ perspective: 1000 }}>
  <div style={{
    transformStyle: 'preserve-3d',
    transition: 'transform 0.6s ease-in-out',
    transform: flipped ? 'rotateY(180deg)' : 'rotateY(0deg)',
  }}>
    <div style={{ backfaceVisibility: 'hidden', position: 'absolute', inset: 0 }}>
      {/* front */}
    </div>
    <div style={{ backfaceVisibility: 'hidden', transform: 'rotateY(180deg)', position: 'absolute', inset: 0 }}>
      {/* back */}
    </div>
  </div>
</div>
```
