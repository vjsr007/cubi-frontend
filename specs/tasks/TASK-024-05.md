# TASK-024-05: Xbox cloud service — Game Pass catalog + UWP installed detection

## Context
Fetches the public Xbox Game Pass catalog (PC Game Pass) and cross-references with locally installed UWP packages / `C:\XboxGames\` directory to mark which titles are installed.

## Parent Design
- **DES**: [DES-024 — Store Cloud Library Design](../design/DES-024-store-cloud-library.md)

## Acceptance Criteria
- [ ] Game Pass PC catalog fetched from public endpoint (no auth required)
- [ ] Installed UWP packages enumerated from Windows registry: `HKCU\SOFTWARE\Classes\Local Settings\Software\Microsoft\Windows\CurrentVersion\AppModel\Repository\Packages`
- [ ] `C:\XboxGames\` directory also checked for installed titles
- [ ] Each catalog title marked `installed: true/false`
- [ ] Uninstalled titles get `file_path = "ms-windows-store://pdp/?productid=<id>"`
- [ ] Results cached with store = `"xbox"`, TTL = 24 h
- [ ] Works on non-Windows (cfg guard: returns empty vec on Linux/macOS)

### Catalog endpoint (public, no auth)
```
GET https://catalog.gamepass.com/sigls/v2
  ?id=fdd9e2a7-0fee-49f6-ad69-4354098401ff   ← PC Game Pass
  &language=en-US
  &market=US

# Returns list of product IDs, then resolve details:
GET https://displaycatalog.mp.microsoft.com/v7.0/products?bigIds=<csv>&market=US&languages=en-US&MS-CV=...
```

### UWP detection (Windows only, cfg(windows))
```rust
use winreg::{enums::*, RegKey};
// Enumerate HKCU\...\AppModel\Repository\Packages
// Look for publisher = "Microsoft Studios" or package family matching Xbox game patterns
// Also scan C:\XboxGames\ subdirectories
```

### Box art
From `displaycatalog` response: `product.LocalizedProperties[0].Images` where `ImagePurpose == "BoxArt"` or `"Poster"`.
