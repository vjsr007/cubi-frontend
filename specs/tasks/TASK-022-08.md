# TASK-022-08: Download URL configuration + browser open

## Context
Allow users to configure a download URL per system, and open that URL in the system browser when clicking a missing game's download button. Also support DAT sync from remote URLs.

## Parent Design
- **DES**: [DES-022 — Game Catalog Database](../design/DES-022-game-catalog-database.md)

## Acceptance Criteria
- [x] "⚙ URL" button in detail toolbar opens a prompt to set the download base URL for current system
- [x] URL stored via `setCatalogDownloadUrl` in TOML config
- [x] Missing games show a ⬇ DL button when a download URL is configured
- [x] Clicking ⬇ DL opens the URL in the system browser via `@tauri-apps/plugin-shell` `open()`
- [x] "Sync" button in detail toolbar triggers `syncCatalog()` to download + import DAT from URL
- [x] "Sync All" button on overview triggers sequential sync for all systems with configured/default URLs
- [x] Default DAT URLs provided for 27 systems via `getDefaultDatUrls()`
