# TASK-024-07: Frontend — installed badge, refresh button, Steam ID input

## Context
Updates `PcGamesPage.tsx` to render the `installed` flag as a colored badge on each game row, adds a "Refresh library" button per tab, adds a Steam ID / API key input in the Steam tab, and makes clicking an uninstalled game open the store's install URL instead of trying to launch the game.

## Parent Design
- **DES**: [DES-024 — Store Cloud Library Design](../design/DES-024-store-cloud-library.md)

## Acceptance Criteria
- [ ] Games with `installed: false` show a grey `NOT INSTALLED` badge
- [ ] Games with `installed: true` show a green `INSTALLED` badge (optional — only if space allows)
- [ ] "Refresh library" button appears in the tab header next to the scan button
- [ ] Clicking "Refresh library" calls the import command with `force_refresh: true`
- [ ] Steam tab shows `Steam ID` text input and `Steam API Key` input in the settings card
- [ ] Save-to-library checkbox is disabled (greyed out) for games with `installed: false`
- [ ] Clicking an uninstalled game row calls `api.openUrl(game.file_path)` via `tauri-plugin-shell`
- [ ] Tooltip on uninstalled game row: "Click to open install page in [Steam/Epic/GOG/Xbox]"

### Badge styles
```tsx
// Not installed
const notInstalledBadge: React.CSSProperties = {
  padding: '2px 7px',
  borderRadius: 4,
  fontSize: 10,
  fontWeight: 700,
  letterSpacing: '0.06em',
  textTransform: 'uppercase',
  background: 'rgba(255,255,255,0.07)',
  color: 'rgba(255,255,255,0.4)',
};

// Installed
const installedBadge: React.CSSProperties = {
  ...notInstalledBadge,
  background: 'rgba(16,124,16,0.2)',
  color: '#4ece4e',
};
```

### openUrl helper
```typescript
import { open } from '@tauri-apps/plugin-shell';
export const openUrl = (url: string) => open(url);
```

### Steam config fields
Add below the SteamGridDB key input in the Steam tab header card:
```tsx
<input
  placeholder="Steam ID (numeric or vanity URL)"
  value={steamId}
  onChange={(e) => setSteamId(e.target.value)}
  style={inputStyle}
/>
<input
  placeholder="Steam Web API Key (store.steampowered.com/dev/apikey)"
  value={steamApiKey}
  onChange={(e) => setSteamApiKey(e.target.value)}
  style={inputStyle}
  type="password"
/>
```
Both saved to config via `saveApiKey()` extended to also persist `steam_api_key` and `steam_id`.
