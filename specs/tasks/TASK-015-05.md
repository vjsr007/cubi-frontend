## Description
Implement the MobyGames API stub as a fallback metadata source for PC games not found by IGDB.

## Acceptance Criteria
- [ ] `search_game(api_key, title) -> Result<Option<MobyGameData>, String>`
- [ ] Endpoint: `https://api.mobygames.com/v1/games?title={encoded_title}&platform=3&api_key={key}`
  - Platform 3 = Windows
- [ ] Rate limiting: 1000ms sleep between requests (free tier = 360 req/hour)
- [ ] `MobyGameData` struct: description, developer, publisher, year, genre, moby_score (0.0-5.0 → normalize to 0.0-1.0)
- [ ] `fetch_screenshots(api_key, moby_id) -> Result<Vec<String>, String>`
  - Endpoint: `https://api.mobygames.com/v1/games/{id}/screenshots?platform=3`
  - Returns up to 5 screenshot URLs
- [ ] Title normalization consistent with IGDB service (share utility function in `services/mod.rs`)
- [ ] Returns `None` if API key not configured (non-fatal, orchestrator skips this source)
