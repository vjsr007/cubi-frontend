# TASK-018-01: Backend Tauri Commands for Metadata + Media Editing

## Context
Create the new Tauri commands module `commands/metadata_editor.rs` with commands for updating game metadata, importing media files (local + URL), deleting media, searching YouTube, and downloading YouTube videos.

## Parent Design
- **DES**: [DES-018 — Custom Game Metadata Editor](../design/DES-018-custom-metadata-editor.md)

## Acceptance Criteria
- [ ] `update_game_metadata` command accepts GameInfoPatch, saves to DB, returns updated GameInfo
- [ ] `import_media_file` command copies local file to app data, updates DB field, returns saved path
- [ ] `import_media_url` command downloads from URL to app data, updates DB field, returns saved path
- [ ] `delete_game_media` command deletes media file and clears DB field
- [ ] `search_youtube` command returns list of YoutubeSearchResult
- [ ] `download_youtube_video` command downloads via yt-dlp, saves to app data, updates DB trailer_local field
- [ ] All commands registered in lib.rs

## Implementation Notes

### Files to Create/Modify
| File | Action | Description |
|------|--------|-------------|
| `src-tauri/src/commands/metadata_editor.rs` | CREATE | New commands module |
| `src-tauri/src/services/media_import_service.rs` | CREATE | File copy, URL download, path helpers |
| `src-tauri/src/lib.rs` | MODIFY | Register new commands |
| `src-tauri/src/commands/mod.rs` | MODIFY | Add `pub mod metadata_editor;` |

### Key Code Patterns
- Follow existing command patterns: `State<'_, Database>`, `AppHandle` for paths
- Media save path: `{app_data_dir}/media/{system_id}/{media_type}/{game_stem}.{ext}`
- Use `GameInfoPatch` + `db.patch_game()` for all DB updates
- Reuse `youtube_service::search_trailer` and `youtube_service::download_video`
- Validate image extensions: jpg, jpeg, png, webp
- Validate video extensions: mp4, mkv, webm, avi
