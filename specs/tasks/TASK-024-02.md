# TASK-024-02: Steam cloud service — owned games via Steam Web API

## Context
Fetches the full list of games owned by the user on Steam (including uninstalled), merges with locally-installed ACF list, and returns enriched `PcImportGame` structs with `installed: bool`.

## Parent Design
- **DES**: [DES-024 — Store Cloud Library Design](../design/DES-024-store-cloud-library.md)

## Acceptance Criteria
- [ ] `resolve_steam_id(vanity, api_key)` resolves a vanity URL to a 64-bit SteamID (passthrough if already numeric)
- [ ] `get_owned_steam_games(steam_id, api_key)` calls `IPlayerService/GetOwnedGames?include_appinfo=1`
- [ ] Each returned game has `installed: true` if a matching `appmanifest_<appid>.acf` exists in any Steam library path
- [ ] Uninstalled games have `file_path = "steam://install/<appid>"`; installed games keep the existing exe path
- [ ] Result is written to `pc_cloud_cache` with store = `"steam"`
- [ ] If `steam_id` is None or API key is missing → falls back to local-scan-only (existing behavior)
- [ ] `force_refresh: false` + cache younger than 24 h → returns cache without API call

### Key API call
```
GET https://api.steampowered.com/IPlayerService/GetOwnedGames/v1/
  ?key=<STEAM_API_KEY>
  &steamid=<64BIT_ID>
  &include_appinfo=1
  &include_played_free_games=1
  &format=json
```
Response: `{ "response": { "games": [ { "appid": 730, "name": "Counter-Strike 2", "img_icon_url": "..." } ] } }`

Box art URL: `https://cdn.cloudflare.steamstatic.com/steam/apps/<appid>/library_600x900.jpg`

### `AppConfig` change
Add `steam_api_key: Option<String>` and `steam_id: Option<String>` to `GeneralConfig`.
