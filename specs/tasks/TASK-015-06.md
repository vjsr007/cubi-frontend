## Description
Implement a PCGamingWiki MediaWiki API client to find the wiki page for a PC game and optionally extract structured infobox data (Metacritic score, store IDs). Free, no auth required.

## Acceptance Criteria
- [ ] `find_page(title: &str) -> Result<Option<PcgwResult>, String>`
  - Endpoint: `https://www.pcgamingwiki.com/w/api.php?action=opensearch&search={encoded_title}&limit=3&format=json`
  - Picks best match by title similarity
  - Returns `PcgwResult { page_title, page_url }`
- [ ] `fetch_infobox(page_title: &str) -> Result<Option<PcgwInfobox>, String>`
  - Endpoint: `https://www.pcgamingwiki.com/w/api.php?action=cargoquery&tables=Infobox_game&fields=Steam_AppID,Metacritic,OpenCritic,GOGcom_ID&where=_pageName%3D"{encoded_title}"&format=json`
  - Returns `PcgwInfobox { steam_appid, metacritic_score, opencritic_score }`
- [ ] `PcgwResult` and `PcgwInfobox` structs
- [ ] No auth, respect 1 req/s courtesy rate limit
- [ ] Unit test with mocked JSON response
