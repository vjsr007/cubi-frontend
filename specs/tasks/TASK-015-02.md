## Description
Implement `src-tauri/src/services/steam_store_service.rs` — fetches full metadata and media URLs from the Steam Store API for a given Steam AppID. No auth required.

## Acceptance Criteria
- [ ] `fetch_steam_store(appid: &str) -> Result<Option<SteamStoreData>, String>` implemented
- [ ] Endpoint: `https://store.steampowered.com/api/appdetails?appids={appid}&cc=us&l=en`
- [ ] Handles `"success": false` gracefully (returns `None`)
- [ ] Strips HTML from `detailed_description` (regex or simple tag stripper)
- [ ] Extracts: description, developer, publisher, year (from `release_date.date`), genres joined, metacritic score, tags from categories, website, background_raw URL, up to 5 screenshot URLs, first MP4 trailer URL
- [ ] `download_steam_media(data, dest_dir, game_name) -> Result<SteamMediaPaths, String>` downloads background + screenshots to correct media folders
- [ ] Rate limiting: sleeps 1500ms between calls (use `tokio::time::sleep`)
- [ ] Reqwest client with 15s timeout and Chrome-like User-Agent header
- [ ] Unit test: mock response for AppID 570 (Dota 2, free game)
