# TASK-018-03: Implement Ruffle auto-detection

## Objective
Detect a local Ruffle desktop installation on Windows and integrate it into the Flash launch flow.

## Implementation Summary

### 1. Create Ruffle detector module
**File**: `src-tauri/src/services/ruffle_detector.rs`

The detector checks common install locations for:
- `%LOCALAPPDATA%/Programs/Ruffle/Ruffle.exe`
- `%LOCALAPPDATA%/Programs/Ruffle/ruffle.exe`
- `%ProgramFiles%/Ruffle/Ruffle.exe`
- `%ProgramFiles(x86)%/Ruffle/Ruffle.exe`
- common portable roots such as `C:\Ruffle\`

### 2. Integrate into `launcher_service.rs`
- Register `Ruffle` as the emulator for the Flash system
- Use detector results in both emulator status and launch resolution
- Keep config overrides working as the highest-priority source

### 3. Apply Ruffle-specific launch settings
- `fullscreen` -> `--fullscreen`
- `renderer` -> `--graphics vulkan|gl|dx12`
- `spoof_url` -> `--spoof-url <url>`
