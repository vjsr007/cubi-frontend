# TASK-021-03: Tauri Commands for Steam Integration

## Context
Create steam_integration.rs commands module exposing Steam functionality to the frontend via Tauri IPC.

## Parent Design
- **DES**: [DES-021 — Steam Reviews & Enhanced Metadata](../design/DES-021-steam-reviews-metadata.md)

## Acceptance Criteria
- [x] `search_steam_games(query)` command → Vec<SteamSearchResult>
- [x] `link_steam_game(game_id, steam_app_id)` → saves appid + fetches all data
- [x] `fetch_steam_data(game_id)` → returns cached data, auto-extracts appid from steam:// URLs
- [x] `refresh_steam_data(game_id)` → force re-fetch bypassing cache
- [x] `extract_steam_appid()` helper parses steam://rungameid/{id} and steam_{id} patterns
- [x] All commands registered in lib.rs

## Definition of Done
- [x] Commands callable from frontend via invoke()
- [x] Proper error handling returning Result<T, String>
