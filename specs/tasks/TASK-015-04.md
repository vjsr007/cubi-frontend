## Description
Extend the existing `steamgriddb.rs` service to fetch hero images (1920×620), transparent logos, and background art in addition to the existing cover grid. Works for all PC sources (Steam, Epic, GOG, EA).

## Acceptance Criteria
- [ ] `fetch_hero(api_key, steam_appid, title) -> Option<String>` fetches wide hero image URL
  - Endpoint: `GET /heroes/steam/{appid}` or `/heroes/game/{sgdb_game_id}`
  - Falls back to title search if appid not available
- [ ] `fetch_logo(api_key, steam_appid, title) -> Option<String>` fetches transparent logo PNG
  - Endpoint: `GET /logos/steam/{appid}` or `/logos/game/{sgdb_game_id}`
- [ ] `fetch_background(api_key, steam_appid, title) -> Option<String>` fetches background/wallpaper URL
- [ ] All three functions share internal `resolve_sgdb_game_id(title)` helper to avoid repeated title lookups
- [ ] Downloaded to: `media/pc/hero/`, `media/pc/logo/`, `media/pc/background/`
- [ ] Returns `None` gracefully on API error or missing assets (non-fatal)
- [ ] Existing `fetch_grid_by_steam_appid` and `fetch_grid_by_name` functions unchanged
