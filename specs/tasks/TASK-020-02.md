# TASK-020-02: Integrate context menu disable into App.tsx

## Objective
Import and use the `useContextMenuDisable` hook in the main App component so that the browser context menu is disabled globally across the entire application.

## Implementation Steps

### 1. Import hook in App.tsx
**File**: `src/App.tsx`

Add this import at the top:
```typescript
import { useContextMenuDisable } from './hooks/useContextMenuDisable';
```

### 2. Call hook in App component
In the App component body (after imports, before return):
```tsx
export function App() {
  // Disable browser context menu, but allow in text inputs
  useContextMenuDisable(true);

  return (
    // ... existing app JSX
  );
}
```

### 3. Manual Testing Checklist
- [ ] Right-click on a game card → browser context menu does NOT appear
- [ ] Right-click on library grid → browser context menu does NOT appear
- [ ] Right-click on settings area → browser context menu does NOT appear
- [ ] Right-click on a text input field → context menu CAN appear (or paste works with Ctrl+V)
- [ ] Ctrl+C / Ctrl+V shortcuts still work in text inputs
- [ ] No console errors in dev tools
