## Description
Implement a two-tier web scraping engine: a fast HTML scraper (reqwest + scraper crate) for static pages, and a headless Chrome scraper (chromiumoxide) for JavaScript-rendered pages. Both extract Open Graph metadata, images, and description from any URL.

## New Cargo Dependencies
```toml
scraper = "0.20"
chromiumoxide = { version = "0.7", features = ["tokio-runtime"] }
url = "2"
```

## Acceptance Criteria

### HTML Scraper (`web_scraper/html_scraper.rs`)
- [ ] `scrape_url(url: &str) -> Result<ScrapedPage, String>` using `reqwest` + `scraper` crate
- [ ] `ScrapedPage` struct: `title`, `description`, `og_image`, `og_title`, `og_description`, `images: Vec<String>`, `canonical_url`
- [ ] Extracts Open Graph tags: `og:title`, `og:description`, `og:image`, `og:url`
- [ ] Fallback: `<meta name="description">`, `<h1>`, `<img src>` (filtered: min 200px hint or no size)
- [ ] Converts relative image URLs to absolute using `url` crate
- [ ] Request timeout: 10s, follows up to 3 redirects
- [ ] User-Agent: Chrome-like string

### Headless Scraper (`web_scraper/headless_scraper.rs`)
- [ ] `check_chrome() -> Option<PathBuf>` — detects Chrome/Chromium via `which::which`
  - Checks: `google-chrome`, `chromium`, `chromium-browser`, `chrome` (Windows: checks common install paths)
- [ ] `scrape_js_url(url: &str) -> Result<ScrapedPage, String>` using `chromiumoxide`
  - Launch headless Chrome with: `--no-sandbox`, `--disable-dev-shm-usage`, `--disable-gpu`
  - Navigate, wait for `DOMContentLoaded` + 2s additional wait for JS
  - Evaluate JS to extract Open Graph meta tags and document title
  - Returns same `ScrapedPage` struct
  - Overall timeout: 30s
- [ ] Gracefully returns `Err` if Chrome not found

### Auto-dispatch (`web_scraper/mod.rs`)
- [ ] `scrape(url: &str, prefer_headless: bool) -> Result<ScrapedPage, String>`
  - If `prefer_headless` and Chrome found → use headless scraper
  - Otherwise → use HTML scraper
  - If HTML scraper returns empty description + empty images → retry with headless if available
- [ ] `download_og_image(og_image_url: &str, dest_path: &Path) -> Result<(), String>`
