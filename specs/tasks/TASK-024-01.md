# TASK-024-01: SQLite cache schema + Rust DB helpers

## Context
Adds the `pc_cloud_cache` table and the three helper methods needed by all four cloud fetchers in subsequent tasks.

## Parent Design
- **DES**: [DES-024 — Store Cloud Library Design](../design/DES-024-store-cloud-library.md)

## Acceptance Criteria
- [ ] `pc_cloud_cache` table created on DB init (idempotent `CREATE TABLE IF NOT EXISTS`)
- [ ] `upsert_cloud_cache(store, games: &[CacheRow])` inserts/replaces rows with current timestamp
- [ ] `read_cloud_cache(store, max_age_secs: u64) -> Vec<CacheRow>` returns rows fresher than max_age
- [ ] `clear_cloud_cache(store: Option<&str>)` deletes rows for one store or all stores
- [ ] All helpers return `Result<_, rusqlite::Error>` (no unwrap)

### Schema
```sql
CREATE TABLE IF NOT EXISTS pc_cloud_cache (
    store        TEXT NOT NULL,
    game_id      TEXT NOT NULL,
    title        TEXT NOT NULL,
    box_art      TEXT,
    developer    TEXT,
    publisher    TEXT,
    protocol_url TEXT NOT NULL,
    fetched_at   TEXT NOT NULL DEFAULT (datetime('now')),
    PRIMARY KEY (store, game_id)
);
```

### Helper structs
```rust
#[derive(Debug, Clone)]
pub struct CacheRow {
    pub store: String,
    pub game_id: String,
    pub title: String,
    pub box_art: Option<String>,
    pub developer: Option<String>,
    pub publisher: Option<String>,
    pub protocol_url: String,
}
```
