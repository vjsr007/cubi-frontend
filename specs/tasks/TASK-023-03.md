# TASK-023-03: Launcher Service Integration — Use Selected Emulator

## Context
Modify the Game Launcher service to respect emulator preferences. When launching a game, the service should query the user's emulator choice for that system, validate the emulator is installed, and fallback gracefully if not. This ensures the correct emulator is used for each game.

## Parent Design
- **DES**: [DES-023 — Multi-Emulator Per-System Support Design](../design/DES-023-multi-emulator-system.md)

## Acceptance Criteria
- [ ] `launch_game()` function in `src-tauri/src/services/launcher_service.rs` modified to:
  1. Query `preferences_service::get_preference(system_id)` to get selected emulator
  2. If preference exists, find the matching `EmulatorDef` in registry
  3. If not found or preference is None, use first EmulatorDef for the system (default behavior)
  4. Check if the selected/default emulator is installed (call `find_emulator_in_emudeck()`)
  5. If not installed, log warning and fallback to first installed emulator
  6. If no installed emulator found, return error with helpful diagnostics
- [ ] Selection strategy clearly documented in comments:
  1. User's preference (if set)
  2. First available emulator with detected path (installed)
  3. First registered emulator for system (even if not installed, let OS fail with clear error)
- [ ] Error messages specify why a game failed to launch (e.g., "No installed emulator for Nintendo Switch. Try installing Ryujinx or Yuzu.")
- [ ] Process launch arguments and environment variables updated to pass correct emulator path (no changes to logic, just use selected emulator's path)
- [ ] Fullscreen flag respected (existing `fullscreen` config honored regardless of selected emulator)
- [ ] All Rust files compile without warnings
- [ ] Comprehensive logging at INFO level for emulator selection and fallback decisions
- [ ] Unit tests for selection logic (prefer, default, fallback scenarios)
- [ ] Integration test: launch game → verify correct emulator was executed

## Implementation Notes
**Guidance for the implementor:**

### Current Launch Flow (Reference)
The existing `launch_game()` likely:
1. Takes `game_info: &GameInfo` or `(system_id, rom_path)`
2. Finds default emulator in registry (first match)
3. Detects emulator path from EmuDeck
4. Builds launch command with template
5. Spawns process with `tokio::process::Command`

```rust
async fn select_emulator(
    system_id: &str,
    app_handle: &tauri::AppHandle,
    app_config: &AppConfig,
    emudeck_path: &str,
) -> Result<EmulatorDef, String> {
    let registry = get_emulator_registry();
    let db = get_db_connection(app_handle)?;
    
    // Priority 1: User's stored preference
    if let Ok(Some(preferred_name)) = preferences_service::get_preference(system_id, &db) {
        log::debug!("Found preference for {}: {}", system_id, preferred_name);
        
        if let Some(def) = registry.iter().find(|d| d.name == preferred_name && d.system_ids.contains(&system_id)) {
            // Check if installed
            if let Ok(Some(path)) = find_emulator_in_emudeck(&def, emudeck_path) {
                log::info!("Using preferred emulator: {} ({})", def.name, path);
                return Ok(def.clone());
            } else {
                log::warn!("Preferred emulator {} not installed for {}, falling back", preferred_name, system_id);
            }
        }
    }
    
    // Priority 2: First installed emulator for system
    for def in registry.iter().filter(|d| d.system_ids.contains(&system_id)) {
        if let Ok(Some(path)) = find_emulator_in_emudeck(&def, emudeck_path) {
            log::info!("Using first installed emulator: {} ({})", def.name, path);
            return Ok(def.clone());
        }
    }
    
    // Priority 3: First registered emulator (may not be installed)
    if let Some(def) = registry.iter().find(|d| d.system_ids.contains(&system_id)) {
        log::warn!("No installed emulator found for {}, using first registered: {}", system_id, def.name);
        return Ok(def.clone());
    }
    
    // Error: No emulator found
    Err(format!("No emulator registered for system: {}", system_id))
}
```
